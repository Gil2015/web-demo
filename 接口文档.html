商城 接口文档

注：接口文档里的url根路径一般都省略的，如http://www.dogMall.com/api/mall/getLists，会直接写/api/mall/getLists

1.1 登录页

1.1.1 登录

请求地址URL

/api/login

请求方法Method

POST

接口描述Description

用户登录

请求接口所需参数 Request Param List

参数名Name 参数含义 Type 是否可以为空
userName 登录用户名 string 否
password 登录密码 string 否

接口返回数据参数 Response Param List

参数名name 参数含义 参数类型Type
callStatus 接口请求状态 string
message 接口请求返回信息 string
token 用户标识 string



2.1 商城首页

2.1.1 商品列表

请求地址URL

/api/mall/getLists

请求方法Method

GET

接口描述Description

获取商城首页狗狗列表数据

请求接口所需参数 Request Param List

无

接口返回数据参数 Response Param List

参数名name 参数含义 参数类型Type
callStatus 接口请求状态 string
message 接口请求返回信息 string
data 数据 array
---id 列表id number
---name 商品名 string
---price 价格 number
---buyNum 已卖数量 number
---imgUrl 商品图片路径 string
---dogNum 商品编号 number
currentPage 当前请求数据是第几页 number
numberPerPage 当前请求数据返回了几条 number
totalNumber 一共有多少条数据 number
totalPage 一共多少页 number
